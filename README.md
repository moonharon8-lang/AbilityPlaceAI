# AbilityPlaceAI
A specialized AI-powered chatbot designed to support people with special needs
> ## AbilityPlace AI ♿
> **A dedicated platform for supporting people with special needs.**
> ### 🚀 Features:
>  * **AI-Powered Chatbot:** Integrated assistant for seamless navigation.
>  * **Accessibility First:** Designed with high contrast and screen-reader compatibility.
>  * **Modern Stack:** Built using C#, ASP.NET, and Supabase for database management.
> ### 🛠️ Technologies:
>  * **Backend:** ASP.NET Core, C#
>  * **Database:** SQL Server / Supabase
>  * **Tools:** Landbot AI, Visual Studio 2022
